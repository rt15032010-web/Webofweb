# Webofweb

Webofweb is an AI-powered website builder project.

Users only need to write a simple description of the website they want, and Webofweb will generate a website automatically.

## Example
User writes:
"Create a modern website for my coffee shop"

Webofweb generates:
- Homepage
- Design
- Sections
- Basic layout

## Features
- Website generation from text descriptions
- Beginner-friendly interface
- Fast website creation
- Mobile responsive design
- AI-powered future plans
